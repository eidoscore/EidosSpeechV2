## Content Format Guidelines

### Structure
1. **Hook** - Compelling intro (2-3 sentences)
2. **Problem** - What problem does this solve?
3. **Solution** - Step-by-step guide
4. **Examples** - Code samples, screenshots
5. **Best Practices** - Tips and recommendations
6. **Conclusion** - Summary + CTA

### Writing Style
- Conversational but professional
- Short paragraphs (2-4 sentences)
- Bullet points for lists
- Code blocks with syntax highlighting
- Screenshots with annotations
- Real-world examples

### CTAs
- "Try eidosSpeech for free"
- "Get your free API key"
- "Start generating AI voices now"
- "No credit card required"

## Blog Technical Setup

### Required Pages
- `/blog` - Blog homepage with post list
- `/blog/[slug]` - Individual post pages
- `/blog/category/[category]` - Category pages
- `/blog/tag/[tag]` - Tag pages
- `/blog/search` - Search functionality

### Features Needed
- RSS feed
- Sitemap integration
- Related posts
- Author bio
- Social sharing buttons
- Comment system (optional)
- Newsletter signup form
- Code syntax highlighting
- Table of contents
- Reading time estimate

### Tech Stack Options
1. **Static Site Generator** (Recommended)
   - Next.js + MDX
   - Hugo
   - Jekyll
   
2. **Headless CMS**
   - Strapi
   - Ghost
   - Contentful

3. **Simple Markdown**
   - Store posts as .md files
