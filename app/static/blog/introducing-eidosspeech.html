3. Simple REST API

Integration takes less than 5 minutes. Our REST API is designed for developers who want to get things done quickly:

curl -X POST https://eidosspeech.xyz/api/v1/tts \
  -H "X-API-Key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Hello world!",
    "voice": "en-US-JennyNeural"
  }' \
  --output speech.mp3

That's it. No complex SDKs, no authentication flows, no OAuth. Just a simple POST request and you get MP3 audio back.
